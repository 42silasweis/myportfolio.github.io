fancyBox

This is a demonstration. More information and examples: www.fancyapps.com/fancybox/

 

Instructions:

  1. Copy the entire "fancybox" folder into your root directory.
  2. Open the code view for this page.
  3. Copy lines 6 through 148.
  4. Open up the page you want the fancybox effect to be in.
  5. Paste the code you copied inside your page <head> tag before the closing </head> tag.
  6. Come back to this page and in Design view click on and copy one of the images below.
  7. Go back to your page and paste the image into the spot you want it to be located in.
  8. Save the page and then test in Live preview to make sure the image effect works.

Notes:

  • To change a caption first click on the image and then open up Code view. Within the image code change the "title".
  • To change effects just change the image class name in the code. Example: fancybox, fancybox-effects-a, etc..
  • Including a group name within each of the images will allow you to click on an arrow to go to the next or previous image that is located on the same page with the same group name. Example: data-fancybox-group="button"
